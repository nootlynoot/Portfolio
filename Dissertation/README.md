## Name
PRIVACY-PRESERVING PERSONAL DATA MANAGEMENT UTILISING BLOCKCHAIN TECHNOLOGY AND UMBRAL PROXY RE-ENCRYPTION

## Description
A decentralized cryptographic file transfer system that enables secure, intermediary-free file sharing using Python, Solidity, Ethereum, IPFS, and Umbral Proxy Re-Encryption (PRE). Files are encrypted before being stored on IPFS, while Ethereum smart contracts coordinate access and proxy re-encryption allows recipients to decrypt data without exposing plaintext to intermediaries. This project demonstrates the practical integration of blockchain, decentralized storage, and modern cryptography through a fully functional end-to-end prototype.
